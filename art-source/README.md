# Art Source Workspace

Store original working art here before exporting game-ready runtime files into `public/assets`.

Recommended structure:

```text
art-source/
  characters/{fighter}/
  stages/{stage}/
  vfx/
  ui/
```

Source files may be `.psd`, `.aseprite`, `.kra`, `.clip`, `.afdesign`, or another editable art format. Runtime exports should remain optimized PNGs or texture atlases under `public/assets`.

Do not copy or trace another game's production art. All final character, stage, VFX, and UI assets should be original to Olympus Brawler.
