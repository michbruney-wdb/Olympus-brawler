# Runtime Texture Atlases

Production fighter and stage atlases should be exported here.

Expected fighter atlas example:

```text
public/assets/atlases/fighters/zeus.png
public/assets/atlases/fighters/zeus.json
```

Keep atlas frame names aligned with current animation naming so code can transition from loose PNGs without renaming every animation:

```text
zeus/idle_1
zeus/run_1
zeus/quick_1
```
